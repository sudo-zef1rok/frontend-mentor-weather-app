# Weather App

Weather App по **техническому заданию** & **дизайну** с сайта FrontendMentor. Проект создан на `Next.js` и `React`

## Что умеет

- Поиск города с подсказками
- Отображение текущей погоды
- Почасовой прогноз
- Прогноз по дням
- Переключение единиц измерения температуры, скорости ветра и осадков
- Обработка ошибок и статусов
- Адаптивная разметка под мобильные и десктопные устройства

## Стек

- `Next.js`
- `React`
- `CSS`
- `OWM API` (OWM - OpenWeatherMap)

## Запуск проекта

1. Установить зависимости:

```bash
npm i
```

2. Создать `.env` файл:

```env
OWM_API_KEY=your_api_key_here
```

3. Запуск Dev:

```bash
npm run dev
```

4. Готово, можно посмотреть и потыкать

```bash
http://localhost:3000
```

> [!NOTE]
> Также можно забилдить версию под прод:
> ```bash
> npm run build
> ```
> а затем папочку build можно залить под капот `Nginx` или `Apache`, но об этом я здесь писать не буду

## Работа с API

Запросы на получение информации о погоде с API происходят с сервера, путём отправки на сервер следующих запросов:

- `GET /api/geocode` - получает список городов для поиска
- `GET /api/weather` - получает текущую погоду и прогноз

> P.S. я делал на базе бесплатного тарифа OWM
